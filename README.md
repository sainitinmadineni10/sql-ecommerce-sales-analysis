# 📊 E-commerce Sales Analysis using SQL

## 🔍 Project Overview
This project analyzes retail sales data using SQL to evaluate business performance, profitability, and regional trends.

## 🎯 Objectives
- Measure total revenue and profit
- Evaluate profit margin and business efficiency
- Identify loss-making regions, categories, and sub-categories
- Analyze impact of discounts on profitability
- Determine top-performing products and locations

## 🛠 Tools Used
- MySQL
- SQL (Window Functions, Aggregations, CASE)

## 📊 Key Analysis

1. Total Revenue & Profit  
2. Profit Margin Calculation  
3. Regional Profit Analysis  
4. Loss-Making Categories & Sub-Categories  
5. Discount Impact on Profit  
6. Top & Bottom Performing Cities  
7. Top Product in Each Region (Window Function)  
8. Profit Ratio Analysis  
9. State-wise Profitability Analysis  

## 💡 Key Insights
- The business is profitable overall, but certain regions incur losses  
- High discount levels significantly reduce profitability  
- Some sub-categories consistently generate negative profit  
- Profitability varies across cities and states  
- Product demand differs by region  

## 📂 Files
- `analysis.sql` → Contains all SQL queries used for analysis  

## 🚀 Conclusion
This project demonstrates the use of SQL for real-world business analysis, including identifying inefficiencies and providing data-driven insights.
